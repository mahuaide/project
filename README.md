# 技术栈

> vue2 + vuex + vue-router + webpack + ES6/7 + axios + stylus + mock.js + element-UI +jQuery（第三方插件）

## 项目运行

``` bash
# 安装依赖
cd project
npm install

# 启动项目，默认端口8081
npm run dev

# 线上发布
npm run build
